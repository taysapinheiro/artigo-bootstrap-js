Criado em agosto de 2011, o Bootstrap é considerado o melhor framework front-end do mundo. As principais características que alimentam sua fama são:
  • documentação super acessível e legível
  • responsividade
  • comunidade ativa
  • biblioteca de componentes, entre outros.
Falando em fama, uma das linguagens de programação que vem ganhando bastante espaço nas aplicações web é Javascript. Não é em vão que o Bootstrap utiliza vários pluggins em Javascript(jQuery) em seus códigos.

Os pluggins do Bootstrap são ativados e configurados através do HTML, utilizando atributos data. Usando o código a seguir, você desativa a funcionalidade da API do atributo data:

$(document).off('.data-api')

Para desativar um pluggin específico, use o namespace do pluggin e em seguida o namespace data-api, dessa maneira:

$(document).off('.alert.data-api')
Se ficou tentado a conhecer mais sobre a junção de Bootstrap + Javascript, mostrarei como utilizá-los em seu projeto e alguns componentes do Javascript que estão no framework. Para fazer uso do JS compilado em seu projeto, basta apenas utilizar o BootstrapCND antes do fechamento da tag body.

Logo abaixo, segue uma lista de alguns componentes dependentes do Javascript e seguida falarei um pouco sobre cada um, utilizando códigos de exemplos.

  • Alerta

    O ‘alerta’ é um componente que apresenta uma mensagem de forma que chame a atenção do usuário.
    Olha que alerta supimpa!
    Comportamento em JavaScript

    $('.alert').alert() - O alerta fica esperando um evento de click nos elemetos que possuem atributo data-dismiss="alert"

    $().alert('close') - Fecha o alerta e o remove do DOM.

    $().alert('dispose') - Destrói o alerta de um elemento

  • Botões

    Os 'botões' são elementos de ações para formulários, diálogos e etc.
    Comportamento em JavaScript

    $().button('toggle') - Deixa o botão com aparência de desativado

    $().button('dispose') - Destroi o botão de um elemento

  • Collapse

    É um pluggin Javascript, utilizado para esconder e mostrar conteúdo

    Link com href

    Anim pariatur cliche reprehenderit, enim eiusmod high life accusamus terry richardson ad squid. Nihil anim keffiyeh helvetica, craft beer labore wes anderson cred nesciunt sapiente ea proident.
    Comportamento em JavaScript

    $('.collapse').collapse() - Ativa manualmente

    $('#myCollapsible').collapse({ toggle: false }) - Ativa o conteúdo como se ele fosse um elemento 'colapsável'

  • Dropdown

    Feito em cima da biblioteca do Popper.js, o dropdown é um botão que expande uma lista de link e outras coisas.
    Alguma ação Outra ação Alguma coisa aqui
    Comportamento em JavaScript

    $('.dropdown-toggle').dropdown() - Ativa o dropdown

    $().dropdown('update') - Atualiza a posição do dropdown de um elemento

    $().dropdown('dispose') - detroi o dropdown de um elemento

  • Modal

    O modal cria notificações de diálogos no site e outros conteúdos personalizados.
    Título do modal
    Que modal lindo!
    Comportamento em JavaScript

    $('#meuModal').modal({ keyboard: false }) - Define conteúdo como modal.

    $('#meuModal').modal('toggle') - Alterna manualmente a visibilidade do modal

    $('#meuModal').modal('hide') - Esconde o modal manualmente

    $('#meuModal').modal('handleUpdate') - Se a altura do modal for alterada, reajusta manualmente.

    .modal('dispose') - Destrói o modal de um elemento
